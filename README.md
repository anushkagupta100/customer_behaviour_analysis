## Overview

This project demonstrates an end-to-end **data analytics workflow** commonly used in industry. It covers data loading and analysis in Python, data cleaning, SQL-based querying across multiple databases, interactive dashboarding in Power BI, and clear business communication through a written report and presentation.

The goal of the project is to extract actionable insights from raw data and present them in a **recruiter-friendly, business-focused** manner.

---

## Dataset

* Source: Provided business dataset (customer/transactional data)
* Format: CSV / Excel (loaded into Python)
* Size: Medium-sized analytical dataset
* Key Characteristics:

  * Structured data with numerical and categorical features
  * Contains missing values and inconsistencies
  * Suitable for exploratory and descriptive analytics

> Note: Sensitive or personally identifiable information (PII) has been excluded or anonymized.

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python using Pandas
* Performed initial inspection (shape, schema, data types)

### 2. Exploratory Data Analysis (EDA)

* Univariate and multivariate analysis
* Distribution analysis and summary statistics
* Identification of trends, patterns, and anomalies
* Business-focused insights derived from data behavior

### 3. Data Cleaning & Preparation

* Handled missing values (imputation/removal as appropriate)
* Corrected data types
* Removed duplicates and inconsistencies
* Prepared clean datasets for SQL and Power BI usage

### 4. SQL Analysis

* Loaded cleaned data into databases
* Executed analytical queries using:

  * PostgreSQL
  * MySQL
  * SQL Server
* Performed:

  * Filtering and aggregations
  * Joins and subqueries
  * Business KPI calculations

### 5. Dashboard Development (Power BI)

* Built an interactive dashboard for key metrics
* Included slicers and filters for dynamic analysis
* Focused on clarity, usability, and business relevance

### 6. Reporting

* Created a concise analytical report summarizing:

  * Key findings
  * Trends and insights
  * Business implications

### 7. Presentation (Gamma)

* Converted insights into a professional slide deck
* Structured for stakeholder and recruiter presentation
* Emphasis on storytelling and decision-making

## How to Run the Project

### Python Analysis

1. Clone the repository
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Open the Jupyter Notebook and run cells sequentially

### SQL Analysis

1. Import the cleaned dataset into PostgreSQL / MySQL / SQL Server
2. Execute the provided SQL queries

### Power BI Dashboard

1. Open the `.pbix` file in Power BI Desktop
2. Refresh data connections if required
